# Car_Community
# CAR FFEINE - 실시간 자동차 커뮤니티

## 개요
CAR FFEINE은 자동차 애호가들을 위한 실시간 커뮤니티 웹사이트입니다. 이 프로젝트는 사용자들이 자동차에 대한 정보를 쉽게 탐색하고, 서로 실시간으로 소통할 수 있는 플랫폼을 제공하는 것을 목표로 합니다.

## 주요 기능
1. **로그인 및 회원가입**
   - 사용자 인증 및 관리
   - 사용자가 입력한 로그인 정보와 회원가입 정보를 데이터베이스와 대조하여 인증

2. **자동차 관련 용어 게시판**
   - 자동차 관련 용어를 데이터베이스에서 불러와 사용자에게 제공
   - 용어의 상세 설명과 추가 이미지를 제공

3. **실시간 그룹 채팅**
   - 브랜드별 채팅방 제공
   - 사용자들이 실시간으로 대화를 나눌 수 있는 공간 제공

4. **차량 정보 게시판**
   - 다양한 자동차 브랜드와 모델 정보 제공
   - 모델의 상세 스펙을 데이터베이스에서 불러와 사용자에게 제공

5. **자동차 관련 뉴스 게시판**
   - 외부 뉴스 API를 통해 최신 자동차 뉴스 제공
   - 사용자가 최신 뉴스를 쉽게 확인할 수 있도록 함

6. **자유게시판**
   - 자유로운 주제로 게시글을 작성하고, 다른 사용자들과 소통할 수 있는 공간 제공
   - 모든 게시글을 데이터베이스에서 불러와 사용자에게 제공

## 사용된 기술 스택
프론트엔드: HTML, CSS, JavaScript, EJS
백엔드: Node.js, Express
데이터베이스: MySQL
실시간 통신: Socket.io
